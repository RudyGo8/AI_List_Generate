# AI List Generate

> An AI-powered baseline project for cross-border e-commerce listing workflow.

[中文版本](./README.zh-CN.md) | [English Version](./README_EN.md)

## 简介 / Introduction

AI List Generate is a baseline AI project for cross-border e-commerce listing workflow.

本项目面向跨境电商商品上架场景，聚焦于商品信息处理、翻译、OCR、类目匹配与文案生成等核心能力，并支持本地开发模式与 Docker Compose 一键启动。
