# AI Listing Frontend (Vue3)

## Start

1. Copy env template

```bash
cp .env.example .env
```

2. Install deps

```bash
npm install
```

3. Run dev server

```bash
npm run dev
```

## Backend Integration

- API base: `VITE_API_BASE_URL`
- headers:
  - `accesskey`
  - `accesssecret`
  - `x-ai-scene`

You can update auth/scene at runtime in the left settings panel.
